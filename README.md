# Controle de Locadora de Veículos 
Sistema console em C para gerenciamento de locadora de veículos. Ideal para disciplinas de Algoritmos, Lógica de Programação e Estrutura de Dados.

### Funcionalidades
- CRUD completo de Clientes (com CPF, CNH e validade)  
- CRUD de Veículos (placa única, modelo, marca, categoria, valor diária)  
- Cadastro de Categorias (Econômico, SUV, Luxo etc.)  
- Locação e Devolução com validação rigorosa (veículo disponível, CNH válida, limite simultâneo)  
- Cálculo automático de diárias, atrasos e multas  
- Persistência total em arquivos binários (.dat) – leitura ao iniciar, gravação ao sair    
- Relatórios:  
  - Carteirinha do Cliente  
  - Balanço da Frota  
  - Balanço de Movimentações (10 veículos mais locados, faturamento etc.)

### Tecnologias
- Linguagem C (compila em Dev-C++, Code::Blocks, GCC)  
- Structs + vetores + arquivos binários  
- Separação de responsabildades  
- Interface texto puro (estilo DOS)

### Como usar
1. Compile com qualquer compilador C  
2. Execute → os arquivos .dat serão criados automaticamente na primeira execução
